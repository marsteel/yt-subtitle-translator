隐私政策

最后更新：2026年1月8日

概述

YouTube双语字幕翻译助手致力于保护您的隐私。本隐私政策说明了在您使用我们的Chrome/Edge扩展时，我们如何处理您的数据。

数据收集

我们不收集、存储或传输任何个人数据到我们的服务器。我们没有服务器。

本地存储的内容

以下数据存储在您的浏览器本地（使用浏览器的sync storage）：

  • OpenAI API密钥：您的API密钥
  • 目标语言设置：您选择的翻译目标语言

所有数据都存储在浏览器的sync storage中，并通过浏览器同步在您的设备间同步（如果启用）。我们永远无法访问这些数据。

第三方服务

当您使用YouTube双语字幕翻译助手翻译字幕时：

  • 字幕文本直接从您的浏览器发送到OpenAI API
  • 我们不拦截、存储或访问这些数据
  • 请查看OpenAI的隐私政策了解他们如何处理数据

API密钥

您的OpenAI API密钥：

  • 存储在您浏览器的加密存储中
  • 永远不会传输给我们或任何第三方（除了OpenAI）
  • 只能由您设备上的扩展访问

权限说明

扩展请求以下权限：

storage

保存您的设置（API密钥和目标语言）到浏览器本地。

host_permissions

访问以下网站：

  • *.youtube.com, youtu.be: 在YouTube页面上注入字幕翻译功能
  • api.openai.com: 与OpenAI API通信以翻译字幕

数据安全

  • 所有设置都使用浏览器的内置加密存储
  • 与OpenAI的通信使用HTTPS
  • 没有数据传输到我们的服务器（我们没有服务器）

开源

YouTube双语字幕翻译助手是开源的。您可以在GitHub上查看完整的源代码，以验证我们的隐私声明。

政策变更

我们可能会不时更新此隐私政策。任何更改都将发布在此页面上，并更新"最后更新"日期。

联系方式

如有隐私问题或疑虑，请在我们的GitHub仓库上提交issue。
