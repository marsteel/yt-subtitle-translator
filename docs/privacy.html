Privacy Policy

Last Updated: January 9, 2026

Overview

YouTube Bilingual Subtitle Translator is committed to protecting your privacy. This privacy policy explains how we handle your data when you use our Chrome/Edge extension.

Data Collection

We do NOT collect, store, or transmit any personal data to our servers. We do not have servers.

What We Store Locally

The following data is stored locally in your browser using the browser's sync storage:

  • API Key: Your AI provider's API key
  • Target Language Settings: Your selected translation target language
  • API Endpoint: Your custom API endpoint URL (if configured)
  • Model Name: Your selected AI model name

All data is stored locally in the browser's sync storage and synced across your devices via browser sync (if enabled). We never have access to this data.

Third-Party Services

When you use YouTube Bilingual Subtitle Translator to translate subtitles:

  • Subtitle text is sent directly from your browser to your configured AI provider's API (OpenAI, Azure OpenAI, Anthropic, Google Gemini, DeepSeek, or custom endpoint)
  • We do not intercept, store, or have access to this data
  • Please review your AI provider's privacy policy for how they handle data:
    • OpenAI Privacy Policy
    • Microsoft Azure Privacy
    • Anthropic Privacy Policy
    • Google Gemini Terms
    • DeepSeek

API Keys

Your AI provider's API key is:

  • Stored in your browser's encrypted storage
  • Never transmitted to us or any third party (except your configured AI provider)
  • Only accessible by the extension on your device

Permissions Explained

The extension requests the following permissions:

storage

To save your settings (API key, target language, API endpoint, and model name) locally in your browser.

host_permissions

To access the following websites:

  • *.youtube.com, youtu.be: To inject subtitle translation functionality on YouTube pages
  • https://*/*: To communicate with any OpenAI-compatible API endpoint you configure (OpenAI, Azure OpenAI, local LLMs, etc.). The extension only connects to endpoints you explicitly specify in settings.

Data Security

  • All settings are stored using the browser's built-in encrypted storage
  • Communication with OpenAI uses HTTPS
  • No data is transmitted to our servers (we don't have any)

Open Source

YouTube Bilingual Subtitle Translator is open source. You can review the complete source code on GitHub to verify our privacy claims.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated "Last Updated" date.

Contact

For privacy questions or concerns, please open an issue on our GitHub repository.
